# Resume Project

## Description
This project is a personal resume website created using HTML and CSS as part of a university assignment in Web Design and Web Programming.

## Goal
To practice creating and styling a responsive web page using semantic HTML and CSS.

## Technologies Used
- HTML5 (semantic structure)
- CSS3 (Flexbox, responsive design)
- Google Fonts

## Features
- Responsive layout (mobile-friendly)
- Based on Figma design
- Clean and structured CSS with comments
